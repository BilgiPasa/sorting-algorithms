If you are using GCC, use this command in this directory to compile the C code:
    gcc -std=c23 -O3 -Wall -Wextra -Wpedantic -o main main.c sorting_algorithms.c

If you are using Clang, use this command in this directory to compile the C code:
    clang -std=c23 -O3 -Wall -Wextra -Wpedantic -o main main.c sorting_algorithms.c

Use this command in this directory to run the C code:
    ./main

If you are going to compile the C code, make sure that you have GCC or Clang installed.

Even if you have a computer that has an x86 CPU, if you have GCC or Clang installed, I recommend you to compile
the code before running.
