If you are using GCC, use this command in this directory to compile the C files:
    gcc -std=c23 -O3 -Wall -Wextra -Wpedantic -o main main.c sorting_algorithms.c

If you are using Linux, use this command in this directory to run the C files that you compiled:
    ./main

If you are using Windows, use this command in this directory to run the C files that you compiled:
    ./main.exe

If you are going to compile the C files with the command above, make sure that you have GCC installed.

Note: -O3 flag is used for to make the C code faster.
